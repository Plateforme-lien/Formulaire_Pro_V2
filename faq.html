Questions fréquentes

Toutes les réponses à vos questions sur la plateforme de référencement

Inscription et accès

Comment m'inscrire sur la plateforme ?

L'inscription se fait en 3 étapes simples :

  • Saisissez votre email professionnel
  • Recevez un code de vérification par email (valable 15 minutes)
  • Remplissez le formulaire (profession, zone d'intervention, disponibilités...)

⏱️ Temps estimé : 5 à 10 minutes

Pourquoi un code de vérification par email ?

Le code de vérification permet de :

  • Sécuriser votre accès et vos données
  • Vérifier que l'email est valide et que vous en êtes bien le propriétaire
  • Éviter les inscriptions frauduleuses

Le code est valable 15 minutes. Si vous ne l'avez pas reçu, vérifiez vos spams.

Je n'ai pas reçu le code de vérification, que faire ?

Si vous ne recevez pas le code dans les 2-3 minutes :

  • Vérifiez votre dossier spam/courrier indésirable
  • Vérifiez que l'adresse email saisie est correcte
  • Si vous avez déjà rempli le formulaire, vous devez attendre 2 heures avant de redemander un code
⚠️ Limite de sécurité : Pour éviter les spams, vous ne pouvez demander que 3 codes maximum par jour.

Puis-je m'inscrire si je n'exerce pas dans le Grand Est ?

Oui ! Vous pouvez vous inscrire dès maintenant, mais la plateforme est actuellement en phase test sur les département : 54 - Meurthe-et-Moselle, 55 - Meuse, 57 - Moselle, 88 - Vosges.

Si vous exercez ailleurs, votre profil sera créé mais la plateforme ne sera pas enore diffusée pour les particuliers.

Modification et gestion de mon profil

Comment modifier mes informations ?

Pour modifier vos informations :

  • Retournez sur le formulaire
  • Saisissez votre email et recevez un nouveau code
  • Vos informations actuelles s'afficheront automatiquement
  • Modifiez ce que vous souhaitez et validez

Vous pouvez modifier vos informations autant de fois que nécessaire.

Comment supprimer mon profil ?

Pour supprimer définitivement vos données :

  • Connectez-vous au formulaire avec votre email
  • Cliquez sur le bouton "Supprimer mes informations" en bas du formulaire
  • Confirmez la suppression
⚠️ Action irréversible : Une fois supprimées, vos données ne pourront pas être récupérées. Vous devrez vous réinscrire si vous changez d'avis.

Comment mettre à jour mes disponibilités ?

Vous pouvez mettre à jour vos disponibilités à tout moment en modifiant votre profil :

  • "Oui, j'ai des places" : Vous êtes visible sur la plateforme
  • "Non, je suis complet" : Vous n'apparaissez plus dans les résultats de recherche
Astuce : Pensez à mettre à jour vos disponibilités régulièrement pour éviter les demandes inutiles.

Zone d'intervention

Comment définir ma zone d'intervention ?

Selon votre mode d'intervention :

  • Cabinet uniquement : Indiquez simplement votre commune d'exercice
  • Déplacement à domicile (Métropole) : Choisissez un temps de trajet
  • Déplacement à domicile (Outre-mer) : Choisissez un rayon en km

La plateforme calcule automatiquement les communes couvertes en fonction de votre temps de trajet ou rayon.

Puis-je couvrir plusieurs départements ?

Oui ! Si vous intervenez à domicile, la plateforme calcule automatiquement les communes accessibles, même si elles sont dans un département limitrophe.
Vous pouvez aussi ajouter manuellement des communes dans n'importe quel autre département.

Puis-je modifier manuellement la liste des communes ?

Oui ! Après le calcul automatique, vous pouvez :

  • Supprimer des communes que vous ne souhaitez pas couvrir
  • Ajouter des communes spécifiques

Astuce : Cliquez sur "Modifier la liste" après le calcul pour personnaliser votre zone.

Vérification et validation

Pourquoi demandez-vous mon numéro SIRET ?

Le numéro SIRET permet de vérifier que vous êtes bien un professionnel en activité. C'est une garantie pour les particuliers.

La vérification est automatique via l'API SIRET de l'État français. Seuls les professionnels avec un SIRET valide et actif sont visibles sur la plateforme.

Quand serai-je visible sur la plateforme ?

Vous êtes visible pour les particuliers 15 min après avoir valider le formulaire, si :

  • Votre SIRET est valide et actif
  • Vous avez accepté les conditions d'utilisation
  • Vous avez indiqué avoir des disponibilités

Que se passe-t-il si mon SIRET n'est pas valide ?

  • Votre profil ne sera pas visible sur la plateforme

Contact et messages

Comment les particuliers me contactent-elles ?

Les particuliers peuvent vous envoyer un message directement via une interface dédiée: Trouver un Professionnel. Vous recevrez leur message par email avec leur adresse de contact.

Vous pouvez ensuite répondre directement par email au particulier (en cliquant sur "Répondre").

Combien de messages puis-je recevoir ?

Pour éviter la saturation et si vous choissiez d'être contacter via un formulaire de contact, la plateforme limite le nombre de messages que vous recevez :

  • Maximum = Nombre de places disponibles × 2 par semaine
  • Exemple : Si vous avez 3 places, vous recevrez maximum 6 messages/semaine

Une fois la limite atteinte, les particuliers verront un message leur demandant de réessayer plus tard.

Si vous avez choisi d'afficher vos informations de contact directement pour les particuliers, cette limite ne s'applique pas à vous.

Que faire si je reçois trop de demandes ?

Si vous êtes saturé de demandes :

  • Mettez à jour vos disponibilités
  • Réduisez le nombre de places disponibles
  • Actualiser vos informations où vous pouvez choisir de ne pas laisser votre mail visible et d'être contacté par formulaire

Données et confidentialité

Mes données sont-elles protégées ?

Oui, la plateforme est conforme au RGPD (Règlement Général sur la Protection des Données) :

  • Vos données ne sont jamais vendues ou partagées à des tiers
  • Elles sont utilisées uniquement pour la mise en relation avec les particuliers
  • Vous pouvez modifier ou supprimer vos données à tout moment

Pour plus de détails : Politique de confidentialité

Combien de temps mes données sont-elles conservées ?

Vos données professionnelles (nom, prénom, email, profession, zone d'intervention, disponibilités) sont conservées tant que vous êtes inscrit sur la plateforme.

Si vous n'avez aucune activité pendant 2 ans, vos données sont supprimées.

Quelles informations sont visibles par les particuliers ?

Les particuliers voient uniquement :

  • Votre prénom et nom
  • Votre profession
  • Les publics que vous accompagnez (tranches d'âge)
  • Votre zone d'intervention (communes)
  • Vos disponibilités
  • Votre mode de contact (si renseigné)

Problèmes techniques

Le formulaire ne s'envoie pas, que faire ?

Si vous rencontrez un problème lors de l'envoi :

  • Vérifiez que tous les champs obligatoires (*) sont remplis
  • Vérifiez votre connexion internet
  • Essayez avec un autre navigateur (Chrome, Firefox, Safari)
  • Videz le cache de votre navigateur

Le calcul des communes ne fonctionne pas

Si le calcul automatique ne fonctionne pas :

  • Vérifiez que votre commune est bien sélectionnée dans la liste déroulante
  • Attendez quelques secondes, le calcul peut prendre un peu de temps
  • Si le calcul échoue, vous pouvez saisir manuellement vos communes
